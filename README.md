## 运行

要部署这个项目，每一个目录都是独立的，只需要简单的go run等命令运行即可
如果遇到服务器项目，大概率是下面这么简单的流程
启动server
```
./server
```
启动客户端
```
./client
或./client -ip 127.0.0.1 -port 8888
或监听 nc 127.0.0.1 8888
```
